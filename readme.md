# Mini Soket

### Learn Websoket with api and html

### Clone the repositori

```
 git clone https://github.com/MRoyhanF/Mini-Soket.git
```

### Install Libraries

```
npm install
```

### Running The Program

```
npm run dev
```

### Test to hit the Endpoint

True Response

```
http://localhost:3000/api

Body :
{
    "message": "true"
}
```

Flase Response

```
http://localhost:3000/api

Body :
{
    "message": ""
}
```
